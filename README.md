# spk-smart-student-learning-assessment
Decision Support System for Student Learning Assessment using PHP, MySQL, and SMART Method.
This project is a web-based Decision Support System developed to assist schools in evaluating student learning progress using the SMART (Simple Multi Attribute Rating Technique) method.

Features
Login Authentication
Dashboard
Student Management
Criteria Management
Alternative Management
SMART Calculation
Ranking Result
Report Generation
Technology
PHP Native
MySQL
HTML5
CSS3
Bootstrap
JavaScript
SMART Method

The system calculates student scores using the SMART method by:

Weight Assignment
Normalization
Utility Calculation
Final Ranking
